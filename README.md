# asterisk
built using AI for coms 4900 at Iowa State University

The main file to run is index.html the other two versions of the index file are previous versions left in to show the larger leaps in changes that were made to the project

Asterisk is a minimalist arcade-style avoidance game where the player navigates a continuous trail through a fixed field of obstacles. Each level presents a dense arrangement of 2×2 asterisk blocks and a single opening in the right wall. The player automatically advances horizontally while controlling vertical movement—holding Enter makes the trail rise, and releasing it causes the trail to fall. The objective is to guide the head of the trail through the opening without colliding with obstacles or the boundaries. Upon success, the player is reset to the start of a new level with increased obstacle density and a newly generated path to solve.

At the start of each level, a short countdown gives the player time to visually plan a safe route through the static obstacle layout. The trail left behind the player persists during each run, creating a clear visual record of movement and adding to the challenge of precise navigation. The game progressively increases in difficulty, emphasizing spatial awareness, timing, and route optimization. Players aim to reach the highest level possible without crashing, with each level introducing tighter spaces and more complex paths.
